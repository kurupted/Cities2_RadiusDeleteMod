# Radius Delete: Bulk Bulldoze (BETA)
A radius bulldoze tool that supports object type filters.
## Features:
- Adjustable radius size
- Filters for object types (eg roads, buildings, trees, props)
- Compatible with both the vanilla bulldozer and Better Bulldozer.
## How to Use:
Open the bulldoze tool and there will be a new radius option.

IMPORTANT: This is a beta version -- use at your own risk. Risk includes corruption of your save file.
