# Radius Delete: Bulk Bulldoze
A radius bulldoze tool with object type filters.

## Features:
- Adjustable radius size
- Filters for object types (roads, buildings, trees, props, surfaces)
- Compatible with both the vanilla bulldozer and Better Bulldozer.

## How to Use:
Open the bulldoze tool and there will be a new radius option.
